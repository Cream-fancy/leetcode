## 70. 爬楼梯

### 题目

假设你正在爬楼梯。需要 *n* 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意：** 给定 *n* 是一个正整数。

**示例 1：**

```
输入：2
输出：2
解释：有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
```

**示例 2：**

```
输入：3
输出：3
解释：有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```

### 题解

动态规划入门题，设 `dp[n]` 表示 `n` 阶楼梯的不同方法数量，得到递推关系式如下：

```cpp
dp[n] = n;                // n = 1, 2
dp[n] = dp[n-1]+dp[n-2];  // n = 3, 4, 5, ...
```

```cpp
int climbStairs(int n) {
  if(n<=2) return n;
  int a=1, b=2, t;
  for(int i=3; i<n; ++i){
    t=a+b, a=b, b=t;
  }
  return a+b;
}
```

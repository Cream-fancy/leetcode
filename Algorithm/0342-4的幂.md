## 342. 4的幂

### 题目

给定一个整数，写一个函数来判断它是否是 4 的幂次方。如果是，返回 `true` ；否则，返回 `false` 。

整数 `n` 是 4 的幂次方需满足：存在整数 `x` 使得 `n == 4x`

**示例 1：**

```
输入：n = 16
输出：true
```

**示例 2：**

```
输入：n = 5
输出：false
```

**示例 3：**

```
输入：n = 1
输出：true
```

**提示：**

- `-2^31 <= n <= 2^31 - 1`

**进阶：**

你能不使用循环或者递归来完成本题吗？

### 题解

```cpp
bool isPowerOfFour(int n) {
  bitset<32> b(n);
  int cnt0=0, cnt1=0; // 统计 1 的次数，和右边第一个 1 出现前 0 的次数
  for(int i=0; i<32; ++i){
    if(b[i]) ++cnt1;
    else if(!cnt1) ++cnt0;
  }
  return cnt1==1 && !(cnt0&1);  // 0 出现偶数次说明为 4 的幂
}
```

## 477. 汉明距离总和

由于异或运算各个位互不影响，按照位来统计。

例如 `nums` 中第 `i` 位如下：

```
[1,0,1,1,0,1]
```

该位两两间不同的数量为：该位 0、1 数量的乘积 `4*2`

```cpp
int totalHammingDistance(vector<int>& nums) {
  long long res=0;
  int sz=nums.size();
  int n0, n1;
  // 按位遍历
  for(int i=0; i<31; i++){
    n0=n1=0;
    // 统计该位 0,1 的数量
    for(int j=0; j<sz; j++){
      if((nums[j]>>i)&1) ++n1;
      else ++n0;
    }
    res+=n0*n1;
  }
  return res;
}
```
